# TechnoAssistant

Приложение для оценки компьютеров с использованием искусственного интеллекта. Проект объединяет JavaFX интерфейс с backend на Python для анализа характеристик ПК.

## Описание

TechnoAssistant — это инструмент для автоматической оценки стоимости и производительности компьютеров на основе их технических характеристик. Приложение использует модели машинного обучения для предсказания цен и рекомендаций.

## Структура проекта

- **JavaUI/** — графический интерфейс пользователя на JavaFX
- **PythonAI/** — модуль машинного обучения и обработки данных
  - `dataset/` — обработанные данные для обучения
  - `models/` — обученные ML модели
  - `helpers/` — вспомогательные функции
  - `statistics/` — статистика и визуализация данных

## Технологии

- Java + JavaFX — пользовательский интерфейс
- Python — машинное обучение и анализ данных
- ML модели для оценки характеристик ПК
